SurvivalPrep

Frontend Repository
Backend Repository

This is my first group project using React and Springboot as frontend and backend respectively. SurvivalPrep centralizes the discussions within a community in order to prepare and recover from natural disasters by allowing users to create threads and posts so they can stay updated and have a platform to ask for assistance.
